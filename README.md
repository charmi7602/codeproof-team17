# CodeProof

CodeProof is a web application that helps recruiters detect fake developer resumes by analyzing GitHub activity and verifying real coding work.

## Problem
Recruiters often struggle to verify whether developers actually built the projects listed on their resumes. This leads to unreliable hiring decisions.

## Solution
CodeProof analyzes GitHub repositories, commit history, and technologies used to generate a trust score for developers.

## Features
- GitHub profile verification
- Trust score generation
- Detection of inflated claims
- Recruiter dashboard
- Developer dashboard

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python Flask  
API: GitHub API
